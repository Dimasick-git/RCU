# RCU (Ryazha Clock Utility)

RCU — это инструмент с открытым исходным кодом для разгона консолей Nintendo Switch под управлением кастомной прошивки Atmosphere. Он позволяет выполнять продвинутую настройку частот процессора (CPU), графического ядра (GPU) и оперативной памяти (RAM) с помощью встроенных средств конфигурации.

> Из-за особенностей Horizon OS разгон оперативной памяти может привести к повреждению данных на NAND или SD-карте.
> Перед использованием убедитесь, что у вас есть полные резервные копии (бэкапы) NAND, PROINFO, EMUMMC и данных на SD-карте.

## О проекте
RCU предоставляет полный набор инструментов для повышения производительности Nintendo Switch. Проект разработан для бесшовной работы с Atmosphere и предлагает гибкий контроль над частотами и напряжением системы.

## Стандартные частоты
* **CPU:** До 1963 МГц (Mariko) / 1785 МГц (Erista)
* **GPU:** До 1075 МГц (Mariko) / 921 МГц (Erista)
* **RAM:** До 1866/2133 МГц (Mariko) / 1600 МГц (Erista)
* Поддержка повышения и понижения напряжения (Overvolting/Undervolting)
* Встроенный конфигуратор
* Совместимость с большинством homebrew-приложений

## Установка
1. Убедитесь, что у вас установлены последние версии Atmosphere и Ultrahand Overlay.
2. Скачайте и распакуйте пакет RCU в корень вашей SD-карты.
   В сборке overlay дублируется в `switch/.overlays/ryazha-clk.ovl` и `switch/overlays/ryazha-clk.ovl` для совместимости с разными лаунчерами/интеграциями.
3. Если вы используете Hekate, отредактируйте `hekate_ipl.ini`, добавив строку:
   ```
   kip1=atmosphere/kips/rcu.kip
   ```
   При использовании fusee изменения не требуются.

## Настройка
1. Откройте оверлей RCU.
2. Перейдите в меню настроек.
3. Настройте параметры разгона по своему усмотрению.
4. Выберите "Save KIP Settings", чтобы применить конфигурацию.
