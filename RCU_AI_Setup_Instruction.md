# Инструкция по настройке AI-автоисправления для RCU

Я успешно интегрировал систему автоматического исправления ошибок GitHub Actions в ваш репозиторий. Теперь, когда любой workflow (например, `Build Horizon OC Zeus`) завершается неудачей, автоматически запускается AI-агент, который анализирует логи и пытается предложить исправление.

## Что было сделано:
1. **Скрипт AI-агента (`scripts/ai_fixer.py`)**: Использует OpenRouter API для доступа к AI-моделям. Теперь он настроен на использование **бесплатных моделей** через `openrouter/free`.
2. **GitHub Workflow (`.github/workflows/ai_autofix.yml`)**: Срабатывает автоматически при провале основного билда.

## Что вам нужно настроить (Секреты GitHub):

Чтобы всё заработало, вам необходимо добавить следующие секреты в настройках вашего репозитория (**Settings -> Secrets and variables -> Actions -> New repository secret**):

| Секрет | Описание |
| :--- | :--- |
| `OPENROUTER_API_KEY` | Ваш ключ от [OpenRouter](https://openrouter.ai/keys). Это позволит использовать AI-модели. Для бесплатных моделей ключ все равно нужен, но оплата не взимается. |
| `GH_PAT` | (Рекомендуется) Ваш Personal Access Token с правами `repo` и `workflow`. Если его не будет, скрипт попытается использовать стандартный `GITHUB_TOKEN`, но у него может не хватить прав для создания веток и PR. |

## Как это работает:
1. Происходит ошибка в GitHub Actions.
2. Запускается `AI Auto-Fixer`.
3. Он запрашивает у OpenRouter решение, используя бесплатные модели.
4. Если решение найдено, он создаёт новую ветку `ai-fix-<run-id>` и открывает **Pull Request** с исправлением и объяснением.

## Важные замечания:
- **Бесплатные модели**: Скрипт теперь использует `openrouter/free`, который автоматически выбирает доступные бесплатные модели на OpenRouter. Список бесплатных моделей может меняться, но обычно включает такие как `deepseek/deepseek-chat-v3-0324:free`, `meta-llama/llama-4-8b-instruct:free` и другие. Вы можете ознакомиться с актуальным списком на [странице бесплатных моделей OpenRouter](https://openrouter.ai/collections/free-models).
- **Безопасность**: Скрипт настроен так, чтобы быть осторожным с компонентами `Ryazha-Авто` и `VRR`.
- **Модели**: Если вы захотите использовать конкретную платную модель (например, `anthropic/claude-3.5-sonnet`), вы можете изменить `MODEL` переменную в `scripts/ai_fixer.py`.
